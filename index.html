Eikon App Devtools

npm install --save-dev pirasis-l/eikon-app-devtools#0.1.0

Getting start

Very simple, run this in node.js environment 

require('eikon-app-devtools')();

Middlewares

Proxy

Require body-parser middleware to interpret the body of each requests

// run on expressjs with default options
app.use(require('body-parser').json());
app.use(require('eikon-app-devtools').proxy());

// or run on expressjs with your custom options
app.use(require('body-parser').json());
app.use(require('eikon-app-devtools').proxy(options));
Property Type Default value
socketUrl String http://localhost:3000
headers Boolean true A flag to consider whether to pass the response header
customUrlRegExp RegExp null An extra regular expression to consider if the URL should be proxied by this module
